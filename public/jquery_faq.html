When was it purchased, yo?
The first parcel of Turkey Run State Park's 2,382 acres was purchased during Indiana's centennial in 1916 when the State Park system was first established.
Is it Indiana's first state park?
Turkey Run is Indiana's second state park.
Where does that weird name come from, bro?
There are many legends about how Turkey Run got its name. One story says that wild turkeys, finding it warmer in the canyon bottoms, or "runs", would often huddle in these runs to avoid the cold. Pioneer hunters would herd the turkeys through these natural funnels into a central location for an easy harvest.
What did those early pioneers leave behind?
The early pioneers have left traces of their heritage.
What color is it?
